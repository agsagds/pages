Часть 01 · Суть

Что такое LoRA

Определение
LoRA (Low-Rank Adaptation) — метод PEFT, который замораживает веса предобученной модели и добавляет маленькие обучаемые матрицы низкого ранга.
Ключевая идея
Вместо обновления всей матрицы весов \(W\) размером \(d \times k\), изменение \(\Delta W\) аппроксимируется произведением двух тонких матриц:
\[\Delta W = B \cdot A\]
Гипотеза intrinsic rank
Изменения весов, необходимые для конкретной downstream-задачи, лежат в подпространстве очень низкой размерности. Полный \(\Delta W\) избыточен.

Full Fine-Tuning vs LoRA
Full Fine-Tuning Обучаем ВСЕ параметры (7B, 70B…) → огромные GPU, дорого, долго LoRA Обучаем только адаптеры (~0.1%) → 1 GPU, быстро, дёшево
Практические свойства
Нет catastrophic forgetting
Multi-task serving
Чекпоинты в MB, не GB
Zero-latency при merge

Часть 04 · Генезис

История LoRA и PEFT

Таймлайн
pre-2019
Transfer learning = feature extraction (frozen backbone + новая голова) либо full fine-tuning. С ростом трансформеров до миллиардов параметров оба подхода стали неприменимы.
2019
Adapters (Houlsby et al.) — маленькие bottleneck-модули внутри слоёв. Матчат full fine-tuning на NLP-задачах. Начало линии PEFT-исследований.
2021
Prefix Tuning и Prompt Tuning — манипуляция входами/hidden states без изменения весов. Ограничения по scale.
Oct 2021
LoRA (Hu et al., Microsoft, ICLR 2022). Ключевой инсайт: intrinsic rank task-specific weight updates низкий. Моделирование ΔW напрямую как low-rank, без архитектурных ограничений адаптеров, с тривиальным merge.
2022–2023
De facto standard в open-source (Stable Diffusion, LLaMA). QLoRA (Dettmers et al., 2023) демократизировал метод через 4-bit квантизацию.
2024+
Переход к адаптивному rank (AdaLoRA), декомпозиции весов (DoRA) и multi-LoRA serving (композиция адаптеров в runtime для mixture-of-tasks).
Ключевой инсайт LoRA
Авторы заметили, что матрица обновления \(\Delta W\) для конкретной downstream-задачи имеет низкий intrinsic rank. Следовательно, нет необходимости учить полную \(d \times k\) матрицу — достаточно двух тонких множителей.
Почему LoRA победил Adapter'ы
Не требует архитектурных изменений
Тривиально merge'ится → ноль latency
Параллельный путь, а не bottleneck
Горячая смена адаптеров
QLoRA (2023)
Комбинация 4-bit NF4 квантизации базы + LoRA адаптеров + Double Quantization. 65B+ модель на одной 48 GB GPU.

Разделяет weight на magnitude и direction. LoRA только на direction.

DYNAMIC
AdaLoRA
Динамически распределяет rank по слоям по важности. SVD-регуляризация.
Compute-heavy
Важно: иерархия не гарантирует качество
Исследование LoRAFactory (2026): хорошо настроенный LoRA с rank 16–32 часто превосходит AdaLoRA и DoRA. AdaLoRA силён только при жёстком parameter budget (rank < 8). DoRA даёт выигрыш при нестабильной тренировке. Начинать всегда стоит с обычного LoRA.
Часть 06 · Оптимизация

Unsloth: ускорение fine-tuning

Что это
Unsloth — open-source фреймворк оптимизации fine-tuning LLM. Hand-written autograd kernels, memory layouts и fused attention, снимающие overhead стандартного PEFT + Transformers.

Prompt Tuning (Lester et al., 2021)
Soft tokens на входе. Эффективно только на моделях > 1B параметров. На small encoders — слабо.
BitFit (Zaken et al., 2021)
Обучает только bias'ы (~0.1%). Дёшево, но слабо для задач, где нужно менять геометрию эмбеддингов.
(IA)³ Infused Adapter by Inhibiting and Amplifying Inner Activations (Liu et al., 2022)
Векторы-масштабаторы для K, V и FFN-активаций. «Ручка громкости» для нейронов. Легко фьюзится в W.
Для small encoders (MiniLM, BGE)
LoRA и (IA)³ — фавориты для контрастивного retrieval. BitFit и Prompt Tuning не справляются с перестройкой embedding-пространства.

Часть 09 · Case Study

IAB Taxonomy × sentence-transformers × ClickHouse

Задача
Классифицировать веб-страницы по иерархии IAB Content Taxonomy: Tier 1 → Tier 2 → Tier 3 (напр., Automotive → Car Culture → Classic Cars).
База: sentence-transformers
all-MiniLM-L6-v2 (22M) или BGE-small (33M). Превращают текст в плотные векторы. Pre-trained кластеризует по общей семантике, не по IAB.
Проблема full fine-tuning
На узком IAB-датасете маленькая модель забывает общую языковую семантику. Edge cases деградируют.
Обучающий датасет (triplets)

  "anchor": "How to prepare authentic carbonara with guanciale and pecorino romano...",
  "positive": "Italian Cuisine — recipes, ingredients and regional dishes of Italy",

LoRA-интервенция
Freeze base weights MiniLM
Inject LoRA в q/k/v/dense projections, r=16, α=32
Train на triplets с MultipleNegativesRankingLoss
Merge W' = W + B·A → standalone-модель
Замороженные веса сохраняют языковой manifold. Адаптер искривляет локальную геометрию — подтягивает эмбеддинги к IAB-центроидам и отталкивает от смежных tier'ов.
Экспорт
LoRA train → merge W+BA → ONNX export → prod
